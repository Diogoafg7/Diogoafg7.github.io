  **Talent Hive** is a marketplace platform connecting independent artists with local venues — musicians, DJs, and performers on one side, bars and event spaces on the other.

  Built end-to-end with **Next.js**, the project covers both the frontend interface and backend logic: user profiles, search functionality, and the data layer connecting artists to venues.

  This was my most complete personal project at the time — I was responsible for the full application structure, from routing and UI to API routes and data handling.

  **Stack:** Next.js, React, Vercel
